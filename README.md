# نظام إدارة الصيدليات الاحترافي
## Professional Pharmacy Management System

### مواصفات النظام
- **PHP Version:** 7.4 وأعلى
- **Database:** MySQL 5.7 وأعلى
- **Frontend:** Bootstrap 5 + AdminLTE + FontAwesome
- **Language:** PHP Native (بدون Framework)
- **Localization:** دعم كامل للعربية RTL

### المزايا الرئيسية
✅ نظام تسجيل دخول آمن
✅ إدارة الصلاحيات والمستخدمين
✅ دعم تعدد الشركات والفروع
✅ نظام POS متكامل
✅ إدارة المخزون والأدوية
✅ إدارة المبيعات والمشتريات
✅ تقارير شاملة
✅ حماية أمنية عالية

### التثبيت
1. نسخ الملفات
2. إنشاء قاعدة البيانات
3. تشغيل `php -S localhost:8000`
